# Creative Tabs
___

**Properties:**

`{tabLabel}` - The label that will show when
passing the mouse over the tab

`[iconItem] = "minecraft:item_frame"` - The
item that will show on the creative tab

**Format Example:**

``` json
{
    "creativeTabs": [
        {
            "tabLabel": "newTab1",
            "iconItem": "customItems:yourNewItem"
        },
        {
            "tabLabel": "newTab2",
            "iconItem": "minecraft:torch"
        },
        {
            "tabLabel": "newTabN"
        }
    ]
}
```
